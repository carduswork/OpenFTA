Preferences Dialog

Control Type
Dialog

Function
Controls the look of the FTA canvas.

Location
Invoked from Preferences...,.

Operation
The dialog consists of:

  • A View panel that specifies whether the canvas should be drawn in colour or monochrome.
  • A Symbol Colours panel that contains a button for each of the symbols. Clicking on any of the buttons launches a colour selection dialog that allows the colour of that symbol to be changed.
  • A Pen Colour panel. Clicking here launches a colour selection dialog that allows the colour of lines and text to be changed.
  • A Highlight Colour panel. Clicking here launches a colour selection dialog that allows the colour of the lines and text of the highlighted symbol to be changed.
  • A Background Colour panel. Clicking here launches a colour selection dialog that allows the colour of the canvas background to be changed.
On activation of the OK button, the current settings are confirmed and the dialog is removed from the screen.
On activation of the Defaults button, the current settings revert back to the original colours set when OpenFTA was first installed.
On activation of the Cancel button, the current settings are discarded and the dialog is removed from the screen.
On activation of the Help button a dialog containing context sensitive help information is displayed.
