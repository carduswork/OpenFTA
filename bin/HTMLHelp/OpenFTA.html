Open (OpenFTA)

Control Type
Menu Option

Function
Opens a previously stored fault tree for display and possible modification.

Location
OpenFTA menubar -> File -> Open

Shortcuts
Accelerator: Ctrl+O.
Toolbar:

Pre-conditions
None.

Operation
Used to open a fault tree that has been previously stored. On activation, a dialog box titled Open is displayed. For more details on the use of this dialog read the section titled File Selection Dialog.
Note that on activation of the OK button, the currently selected file is opened and the Open dialog is removed. If files containing data in an incorrect format are selected then, an error dialog is displayed.
If a fault tree has been modified without saving, then confirmation is requested before OpenFTA proceeds with the Open Operation.
On successful completion of the operation, the newly chosen fault tree is displayed.
