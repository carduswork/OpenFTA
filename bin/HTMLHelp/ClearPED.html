Clear (OpenPED)

Control Type
Push Button

Function
Provides a means to clear the fields in the primary events database window.

Location
OpenPED -> Cancel

Shortcuts
Accelerator: None.
Toolbar: None.

Pre-conditions
None.

Operation
On activation, this causes the ID and Description text fields to be cleared, the Type radio buttons to be deselected and the Constant Failure Rate/Unit Time and Dormant toggle buttons to be deselected and the Probability set to zero.

Cross References
Apply (OpenPED).
