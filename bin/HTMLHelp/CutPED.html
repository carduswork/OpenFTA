Cut (OpenPED)

Control Type
Menu Option

Function
Removes the current selected text from the ID, Probability or Description fields and copies it to the clipboard.

Location
OpenPED menubar -> Edit -> Cut

Shortcuts
Accelerator: Ctrl+X.
Toolbar:

Pre-conditions
Text is selected in one of the text fields.

Operation
Removes the current selected text from the IID, Probability or Description fields and copies it to the clipboard.
Selections copied to the clipboard are available for the Paste operation. Any information previously held on the clipboard is lost when a Cut operation is carried out.
