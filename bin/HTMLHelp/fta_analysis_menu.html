Analysis

Control Type
Pulldown Menu

Function
Provides access to facilities for analysing fault trees.

Location OpenFTA menubar -> Analysis

Shortcuts
Accelerator: None
Toolbar: None

Pre-conditions
None

Operation
This menu provides the following options:

  • Validate - determines if the fault tree currently displayed is valid and hence ready for further analysis
  • Minimal Cut Sets... - displays a dialog from which minimal cut set generation is initiated
  • Numerical Probability... - displays a dialog from which a deterministic qualitative analysis of the fault tree is initiated
  • Monte Carlo Simulation... - displays a dialog from which a statistical analysis of the fault tree is initiated.
