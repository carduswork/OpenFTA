View

Control Type
Pulldown Menu

Function
Provides access to facilities for displaying different views of the fault tree.

Location
OpenFTA menubar -> View

Shortcuts
Accelerator: None.
Toolbar: None.

Operation
Tree Overview shows the entire tree in a window. Zoom facilities allow inspection of an area of the tree in greater or lesser detail.

Pre-conditions
None.
