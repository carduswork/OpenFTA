Print... (OpenFTA)

Control Type
Menu Option

Function
Prints the currently displayed fault tree.

Location
OpenFTA menubar -> File -> Print...

Shortcuts
Accelerator: Ctrl+P.
Toolbar:

Pre-conditions None.

Operation
Used to print the fault tree. On activation, a dialog titled Print is displayed. For more details on the use of this dialog read the section entitled Print Dialog.
