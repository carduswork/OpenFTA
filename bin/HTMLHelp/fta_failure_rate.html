Constant Failure Rate/Unit Time

Control Type
Check box

Function
Provides a means to state a primary event's failure rate in tems of the probability that it wil fail within a given time period i.e. as a function of time (λ-model) rather than as an absolute probability (p-model).

Location
OpenPED -> Constant Failure Rate/Unit Time

Shortcuts
Accelerator:None.
Toolbar:None.

Pre-conditions
None.

Operation
If set, on activation of the Apply button, the probability of the current event will be entered into the primary events database as a function of time (λ-model), rather than as a function of probability, (p-model).
