Save As... (OpenFTA)

Control Type
Menu Option

Function
Stores the current fault tree to a named file.

Location
OpenFTA menubar -> File -> Save As...

Shortcuts
Accelerator: Ctrl+A.
Toolbar: None.

Pre-conditions
None.

Operation
Used to store the open fault tree to a named file. The file name is usually of the form <name of tree>.fta.
On activation a dialog box titled Save, is displayed. For more details on the use of this dialog read the section titled File Selection Dialog.
Note that on activation of the Save button the fault tree is stored to the currently selected file and the Save dialog is removed.
If the output file does not have write access, then an error dialog is displayed. In order to save the fault tree the permissions must be changed or the Save As... operation must be used to store the fault tree to a file with a different name and/or directory location.
