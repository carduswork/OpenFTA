Paste (OpenFTA)

Control Type
Menu Option

Function
Places the clipboard contents as a child of the currently selected fault tree symbol.

Location
OpenFTA menubar -> Edit -> Paste

Shortcuts
Accelerator: Ctrl+V.
Toolbar:

Pre-conditions
A symbol in the drawing area is selected.

Operation
A copy of the clipboard contents, usually a sub-tree of gates and events, is added as a child of the currently selected fault tree symbol. If this operation would cause the fault tree to become invalid, then an error dialog is displayed and the operation is aborted. The Paste operation can be repeated several times, since the clipboard retains its contents until they are overwritten by a Copy or Cut operation.
