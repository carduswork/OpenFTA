Undo

Control Type
Menu Option

Function
Undoes the last edit action carried out upon the fault tree.

Location
OpenFTA menubar -> Edit -> Undo

Shortcuts
Accelerator: Ctrl+Z.
Toolbar: None.

Pre-conditions
One of the edit actions: Copy, Cut, Delete, Paste, Shift Left or Shift Right has been carried out or a symbol has been added to the fault tree.

Operation
On activation, the last edit action or symbol addition to the fault tree is undone. Note that if the Undo option is selected twice in a row it undoes the first undo action.
