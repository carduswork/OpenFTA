Set Zoom...

Control Type
Menu Option

Function
Set’s the zoom value of the OpenFTA canvas.

Location
OpenFTA menubar -> File -> Set Zoom...

Shortcuts
Accelerator: None.
Toolbar: None

Pre-conditions
None.

Operation
Invokes a dialog that allows the current zoom to be set. The zoom can be entered manually or a preset value selected from the drop down list.
