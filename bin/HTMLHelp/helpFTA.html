Help on help

Help pages may be accessed in two ways:

  1. From the Help option in the main menu bar
    This provides a number of Help topics directly, and an Index option. Selecting Index will open your current default internet browser giving a hypertext list of all help topics available.
  2. From the Help buttons in the dialogs
    Pressing HELP in a dialog will cause the relevent Help page to appear. These Help pages are also available through the index.
