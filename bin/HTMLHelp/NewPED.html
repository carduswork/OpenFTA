New (OpenPED)

Control Type
Menu Option

Function
Provides an empty primary events database.

Location
OpenPED menubar -> File -> New

Shortcuts
Accelerator: Ctrl+N
Toolbar:

Pre-conditions
None.

Operation
Used to create an empty primary events database.
If the database currently on display has been modified and not saved, then confirmation is requested before OpenPED proceeds with the New operation.

Cross References
None.
