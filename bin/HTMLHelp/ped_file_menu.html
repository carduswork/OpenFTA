File (OpenPED)

Control Type
Pulldown Menu

Function
Provides access to facilities for manipulating primary events databases.

Location
OpenPED menubar -> File

Shortcuts
Accelerator: None.
Toolbar: None.

Pre-conditions
None.

Operation
Used to invoke the following operations; New, Open..., Save, Save As..., Export Text File..., Printer Setup..., Page Setup..., and Print...
