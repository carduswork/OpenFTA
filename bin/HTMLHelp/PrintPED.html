Print... (OpenPED)

Control Type Menu Option

Function
Prints the currently displayed primary events database either to a printer or to a file.

Location
OpenPED menubar -> File -> Print...

Shortcuts
Accelerator: Ctrl+P.
Toolbar:

Pre-conditions
None.

Operation
Used to print the primary events database either to a connected printer or to a file. On activation, a dialog titled Print is displayed. For more details read the section entitled Print Dialog.
