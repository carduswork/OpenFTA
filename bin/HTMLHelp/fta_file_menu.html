File (OpenFTA)

Control Type
Pulldown Menu

Function
Provides access to facilities for manipulating fault tree files.

Location
OpenFTA menubar -> File

Shortcuts
Accelerator: None.
Toolbar: None.

Pre-conditions
None.

Operation
Used to invoke the following operations; New, Open..., Save, Save As..., Export Image..., Printer Setup..., Page Setup..., Print Preview..., Print..., Database and Exit.
