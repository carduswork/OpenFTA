Export Text File...

Control Type
Menu Option

Function
Exports the OpenPED database as a text file.

Location
OpenPED menubar -> File -> Export Text File..

Shortcuts
Accelerator: Ctrl+E.
Toolbar: None.

Pre-conditions
None.

Operation
Exports the OpenPED database as a text file. A file selection dialog is invoked, allowing the target file to be specified.

Cross References
None.
