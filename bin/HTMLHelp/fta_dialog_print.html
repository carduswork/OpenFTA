Print Dialog

Control Type
Dialog

Function
Prints an image to a printer.

Location
Invoked from Print... (OpenFTA), Printer Setup... (OpenFTA), Print... (OpenPED), Printer Setup... (OpenPED) and Print... (OpenFTA, Report).

Operation
The dialog consists of:

  • a Printer Name list allowing the destination printer to be specified
  • a Properties button that invokes the properties dialog specific to the currently selected printer
  • a Print Range panel allowing a subset of pages to be selected for printing
  • a Copies panel where the number of required copies can be set
On activation of the OK button, the document is sent to the selected printer and the dialog is removed from the screen.
On activation of the Cancel button, the procedure is aborted and the dialog is removed from the screen.
On activation of the Help button a dialog containing context sensitive help information is displayed.
